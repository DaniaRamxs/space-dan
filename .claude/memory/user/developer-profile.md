# Perfil del Desarrollador

## Rol
Desarrollador principal (y probablemente único) de Spacely. Toma todas las decisiones de producto y técnicas.

## Idioma
Español. Toda la comunicación es en español.

## Forma de trabajar
- Comparte console logs del browser para reportar bugs
- Espera que el agente diagnostique y resuelva — no da instrucciones técnicas detalladas
- Trabaja en Windows 10, VSCode con extensión Claude Code
- Usa el dev server en localhost:3001

## Proyecto
Spacely — red social gamificada con temática espacial. Multiplataforma: web, desktop (Tauri) y Android (Capacitor).
